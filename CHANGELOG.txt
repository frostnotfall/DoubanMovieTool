telegram-bot Changelog
========================
1.9.24 - 2019/2/19
- Instant View 优化，当电影或影人未获奖时不显示相关文字
- 修复 urllib 保存 cookie后不会再次加载
- 修复 aiohttp session 传递后提示未关闭的问题

1.9.23 - 2019/2/18
- 换用 ujson 解析，优化性能
- 优化日志输出

1.9.22 - 2019/2/17
- aiohttp 添加 cookie , header
- 修复某些时候不返回电影详情的问题

1.9.21 - 2019/2/16
- 基本性能优化
- 修复没有评分的情况

1.9.19 - 2019/2/15
- “电影、演员详情”弃用之前的图片加文字方式返回结果，改用 Instant View 呈现输出（嗯，舒服了）

1.8.19 - 2019/2/14
- 增加“获奖情况”功能，目前只针对“演员详情”开启
- 修复“演员详情”内容过多时不返回消息，telegram api 限制不超过 1024 个字符

1.8.17 - 2019/2/6
- 增加快捷搜索功能， 使用 Inline mode
- 原“电影搜索”，“演员搜索功能”已整合进“其它搜索方式“按钮中
- 返回值细节优化， 个别电影信息受到限制，直接返回url

1.7.16 - 2019/1/24
- 定时清除过期的词云图片
- 修复BUG：发送消息后返回无响应的提示

1.7.15 - 2019/1/24
- 添加演员搜索功能
- 更改电影详情消息发送方式
